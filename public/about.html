About This Visualizer

This site is a hands-on tool which allows students to learn data structures by visualizing them in action. It was built to serve as an interactive diagram to help students understand and map the concepts to an example implementation.

Who is the tool for?

  • Computer science students studying data structures
  • Educators demonstrating data structure behaviors in class
  • Anyone reviewing foundational data structures

This tool was built with

  • Front-end: HTML, CSS, JavaScript, and SVG
  • Back-end: C++ with the Crow web framework
